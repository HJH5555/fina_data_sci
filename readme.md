任务：构造一个折旧数据的模拟生成器，让玩家可以直观看到折旧数据在时间维度上的变化情况

交互界面：
一个游戏掌机的构造，一个屏幕显示折旧金额在时间维度上的波动，下面有三个按键，中间是开始/停止折旧，左边是更新改造（资本化），右边是减值


核心逻辑：
1. 一个可以根据玩家设置的参数，生成折旧金额的折旧模拟器，可以根据参数，在年限平均，双倍余额递减和年数总和法等不同场景下生成每月折旧的数字
2. 当玩家按下更新改造/减值键，可以按照新的资产价值生成折旧
3. 可视化地展现折旧数据伴随业务发生的波动，让玩家感受到数据模式与实际业务之间的关联性

注意事项：
1. 简洁优雅的设计语言
2. 可视化地表达折旧金额的变化，建议使用平滑的线型图
3. 可以有开始和结束页面，便于展示
